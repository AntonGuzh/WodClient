# История изменений

Все заметные изменения проекта сгруппированы по релизным тегам.

## v1.0.5 - 2026-06-29

### Добавлено

- Добавлена 3D-анимация бросков d10 в модальных окнах проверок.
- Добавлены отдельные frontend-файлы для 3D-роллера кубов и логики dice-модалок.
- Добавлены 3D-кубы голода с красным цветом и отдельными иконками для звериного провала, кровавого успеха и кровавого триумфа.
- Добавлен выбор обычных 3D-кубов для переброса за силу воли прямо на сцене.

### Изменено

- Конкретные значения кубов теперь определяются на frontend по результату 3D-броска, а backend по-прежнему считает количество успехов и тип результата.
- Убраны 2D-иконки результатов под 3D-сценой.

## v1.0.4 - 2026-06-28

### Добавлено

- Добавлен README с описанием проекта, запуском, структурой папок и правилами разработки frontend.
- Добавлены специализации навыков: хранение в модели персонажа, backend API и Alpine-компонент интерфейса.
- Добавлен отдельный Alpine-компонент для трекера человечности.

### Изменено

- Обновлена логика работы человечности и ее отображения в интерфейсе.
- Debug-режим `pywebview` выключен по умолчанию. Для локальной разработки он включается через `WOD_DEBUG=1`.

## v1.0.3 - 2026-06-28

### Добавлено

- Добавлено автосохранение текущего персонажа.
- Добавлены данные и изображения кланов.
- Добавлен frontend API-адаптер `static/js/api.js` между Alpine-компонентами и `pywebview`.
- Добавлены Alpine-компоненты для повторно используемых элементов интерфейса:
  - трекеры значений;
  - модальные окна;
  - список преимуществ и недостатков;
  - список дисциплин;
  - поля ввода текста;
  - текстовые области;
  - трекеры здоровья и воли;
  - выбор клана и информационные селекторы.

### Изменено

- Значительная часть поведения перенесена из inline-скриптов в отдельные Alpine-компоненты.
- Стили вынесены в `static/css/style.css`.
- Переработана работа с кланами, дисциплинами, преимуществами, текстовыми полями и жизненными трекерами.
- Обновлена структура frontend-кода, чтобы компоненты обращались к backend через общий API-слой.

## v1.0.2 - 2026-04-13

### Добавлено

- Добавлено автоподстраивание высоты текстовых областей.

### Исправлено

## v1.0.1 - 2026-04-11

### Исправлено

- Исправлены проблемы с кодировкой на Windows при чтении и записи JSON-файлов.

## v1.0.0 - 2026-04-11

### Добавлено

- Первый релиз приложения.
- Добавлен основной лист персонажа Vampire: The Masquerade / World of Darkness.
- Добавлены модели персонажа, клана, дисциплины и способности дисциплины.
- Добавлены контроллеры для персонажей, кланов, дисциплин и бросков костей.
- Добавлен backend API для связи Python и frontend через `pywebview`.
- Добавлен экран списка персонажей.
- Добавлено создание, открытие и сохранение персонажей в JSON.
- Добавлены базовые данные дисциплин и изображения дисциплин.
- Добавлены изображения результатов бросков костей.
- Добавлена сборка релизных артефактов через GitHub Actions для Windows, macOS и Linux.

### Исправлено

- Исправлен workflow релизов, чтобы сборка по тегам создавала GitHub Release с артефактами.
